This is my opinionated set of AI skills to boost my Laravel projects development.

## Installation

You can install the package via composer:

```bash
composer require alibori/laravel-ai-skills --dev
```

Then run Laravel Boost command to make your AI agents aware of these skills:

```bash
php artisan boost:install
```
